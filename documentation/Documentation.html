Calendar's User Documentation

Event manager — Category manager — Settings — Daemon — Help


Calendar is a native Haiku application to manage your appointments. It was originally developed by Akshay Agarwal for the Google Summer of Code of 2017, see his Blog posts about it. It's further developed and maintained by developers at HaikuArchives.

You can create, edit and delete events, and cancel or hide them, categorize events and show them in a day, week or month view.

At the top left is today's date (click it to jump back to "today" from whenever you are in the calendar). Below is a calendar view, highlighting the currently shown date. Days with events are tinted as well; the more events on a day, the darker they get. Go figure... :)

To the right is the actual list of events. With the tabs at the top, you set the view to show all events for that Day, Week, or Month.

Right-click an event to get a context menu to create a New event, or Edit, Delete, Cancel or Hide an existing event.
Hidden and deleted events can remain visible by activating View | Show hidded/deleted events and are then tinted red. Deleted events are moved to the system's Trash. If you empty the Trash, your events will be truely gone and cannot be "un-deleted" any more!

Event manager

Double-clicking an event also edits an event. As when creating a new event, it opens the Event manager.

The Event manager lets you set all the details of an event:

  • Add a Name, Place and Description.
  • Define as an All day event or set Start/End date and time (click the "▼" button for a pop-up calendar for the date, the time setting is HH:mm (24-hour format)).
  • Optionally choose a monthly or yearly Recurrence (not implemented yet).
  • Set the Status to Cancelled or Hidden (also done with the context menu of the main window).
  • Optionally give it a Reminder.

Category manager

You can create and edit event categories by invoking Category | Manage....

Calendar comes with only two categories preinstalled: "Default" and "Birthday". The "Default" category cannot be deleted but can be modified. Click on New... or select a category and click Edit... to open the Category edit window:

Here you simply enter the name of the category and choose a color for it.

Settings

App | Settings... opens a window with a few available options:

  • Set the Default category for newly created events.
  • Have the First day of the week either to whatever the system locale says, or choose a specific day.
  • Choose to Show week numbers in calendar.

Daemon

Calendar Daemon runs in the background traking the events to give off reminder alerts for the events at specified time (as set by the user).

It was developed under Google Summer of Code 2022 by Harshit Sharma, see his wrap up blog post about it.
